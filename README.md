# STUDIO_FIT ecommerce site

This is a static ecommerce storefront for STUDIO_FIT. Open `index.html` directly for local preview, or deploy the folder to any static host.

Before going live:

- Deploying on Vercel works without a custom domain. Add SEO files such as `robots.txt` and `sitemap.xml` later if you connect a real domain.
- Connect checkout to a real payment processor such as Stripe, PayPal, or your ecommerce backend.
- Connect account/login and newsletter forms to a real backend or email service.
- Replace demo catalog data in `js/catalog.js` with your final products, prices, descriptions, and real product photography when ready.
- Add analytics, privacy policy, terms, shipping, and returns pages for your business.
